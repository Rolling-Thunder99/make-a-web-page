Lesson 1: Basics of the Web and HTML

Major pieces of the Web
World Wide Web is made up of:
  1. The users
  2. PCs and Browser
  3. Internet
  4. Servers
HTML

HTML stands for Hypertext Markup Language. HTML contains these elements:

  1. Text content - what you see
  2. Markup - what it looks like
  3. References to other documents such as:
    • Images
    • Videos
    • Other documents
  4. Links to other pages

There are two kinds of HTML tags:

  1. Opening tag - closing tag. e.g. with content < name > contents < /name >
  2. Void tags - These tags only need an opening tag since they have no content. They reference to other pages, images, etc.
    e.g.
    < br > The line break tag.
The < img tag >. It is used to reference images from other locatons on your html page. The alt text part is used if the image reference is broken. The format of this tag is < img src = "url" alt = "text">

One last tag I will mention now is the anchor tag < a > < /a >. The anchor tag is used to direct you to a website or another document on your own website by a hyperlinked word, phrase, icon, etc. The format of the anchor tag is < a href = "www.somewebpage.com" > some linked word < /a >.

Inline vs Block Elements
Elements are refers to everything within a set of opening and closing tags.

Inline elements

Only takes up as much width as it needs. Does not force new lines.

Examples of Inline Tags:
  1. < b > < /b > is the bold tag.
  2. < em > < /em > is the emphasis tag.
  3. < img > tag is a void tag...no closing tag.
  4. < span > < /span > is the span tag. It changes things on the lines between the span tags.
  5. < strong > < /strong > is the strong tag which is basically an inline bold function.

Block elements

Takes up the full width available, with a new line before and after. It also forces an invisible box around the content within the tags.

Examples of Block Tags:
  1. < p > < /p > is the paragraph tag.
  2. < div > < /div > is the divider tag. It is used to identify the webpage structure and to lay out the web page. Used to group items.
Computers are stupid

Computers only respond to the correct input. So programmers need to write code in the proper syntax for the computers to understand.

Developer Tools inside a Browser

You can inspect elements of the web page inside the browser. Elements can be either inline or block. Think of block elements as having an invisible box around the contents inside of them. All elements are rectangular.

You can modify the webpage from the developer tools without affecting the html or css files associated to it. The changes will only be temporary and can be discarded with a browser refresh.

Tree Like Structure

Lesson3: Adding CSS style to HTML Structure

Inheritance

Inheritance is a mechanism by which properties are applied not only to a specific element but also to its descendents. Descendent elements inherit text-related properties but not box-related properties.

Some Inheritable properties

  1. Color
  2. Font
  3. Line-height

Some Non-Inheritable properties

  1. Background
  2. Border
  3. Display
3 ways to include CSS style in Webpage
  1. Method 1: Write CSS in the <head > section of your HTML.
    Use < head >
    < style >
    div {
    }
    < /style >
    < /head >
  2. Method 2: Write you style inline with your HTML

    This is a bad idea because it lends to lots of repeated code and lots of added work. So avoid this method as much as possible.

  3. Method 3: Link your HTML to a separate CSS file by writing the link address in your < head > section of your HTML.

    This is the method we will use predominately.

    Use < head >
    < link rel="stylesheet" type="text/css" href="main.css" >
    < /head >

    The link rel="stylesheet" portion links the css document to the html as long as it is located in the same folder.

Styling Up

We already linked the CSS document to the HTML document. Now it is time to add the style to the elements of our choice. This can be done to basic elements like < h1 > or defining classes or ids in the HTML with a < div > tag or an < id > tag.

A simple example of adding CSS to an HTML element:

h1 {
background-color : black;
color : white;
}

In this example the h1 is known as the selector. Anything written between the curly braces will be interpreted as a rule and will be applied to all h1 elements.

The line background-color : black; is a declaration. background-color is a property and black is a value of that prperty.

The CSS Box Revisited

The CSS box has 4 levels to its design. Starting from the inner most box layer.

  1. Content: is the layer that holds images or text appearing on websites. It is also the nucleus because it sits at the center of the box.
  2. Padding: is the layer that clears area around the content layer. Its affected by the background color of the box. Protects the contents of the box.
  3. Border: is the layer inherited from color property of the box.
  4. Margin is the layer that clears area around the border. Clears space between boxes. It doesn't have a color. It is completely transparent.

The element is equal to the sum of the following from border-left, padding-left, content, padding-right, and border-right. You can view this box in the dev tools- computed tab.

However there is an easier to use model and we will plan to use this model.

It is CSS styling set up as such in your CSS page.

* {
box-sizing : border-box;
 }

This allows for the calulation of the box to include both border and padding. Just set the width of the overall box and the size of the box stays the same. Margin is not included in this size.

But you need to add browser specific prefixes to the box-sizing : border-box in order for older browser to display this feature. So you should add these declarations to the code:

* {
box-sizing : border-box;
-moz-box-sizing : border-box;
-webkit-box-sizing : border-box;
-ms-box-sizing : border-box; }

Box-sizing choices: Pixels or Percentage

You can set size box sizes as a percentage (%). If you do this the size of the box will change as you resize the box.

You can set size of box as a pixel (px). If you do this the size of the box will not change from your original sizes no matter how the box is resized. This lends to bad user experience.

Positioning Boxes

< divs > are block elements, as opposed to inline elements such as span, so they take up the entire width of a page. If you want two < div > elements to be side by side on a page you need you can change the display attribute of the parent container to value flex. Adding that to the parent element, the child elements align next to each other.

For it to work you have to put in a size that is smaller than 100%. To make it work put display : flex; in your parent element in html under style or put it in your CSS file. Example below for putting in your CSS file:

.app {
display : flex;
}

Adding Images

Adding images takes a few steps:

  1. First make sure you create a folder in your main project folder called something simple like images.
  2. Next put all images in that folder that you will be using on your web page.
  3. Lastly add a line to your HTML code that directs you to the image.

    < div class="image">< img src="images/image.gif" alt = "image" >

Benefits of Avoiding Repetition

The benefits to avoiding repetition

  1. Save time by writing less code for the elements that use the same stle centralized in a CSS file.
  2. Editing is easy because of the central nature of the CSS for the elements
